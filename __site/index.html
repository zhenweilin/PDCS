Overview

This project implements the algorithm for solving conic optimization problems of the form:

min⁡x=(x1,x2),x1∈Rn1,x2∈Rn2c⊤x  s.t.Gx−h∈Kd,l≤x1≤u,x2∈Kp,\min_{x=(x_1,x_2),x_1\in \mathbb{R}^{n_1}, x_2\in \mathbb{R}^{n_2}} c^{\top} x\ \ \text{s.t.} Gx-h\in \mathcal{K}_d, l\leq x_1\leq u, x_2 \in \mathcal{K}_p,x=(x1​,x2​),x1​∈Rn1​,x2​∈Rn2​min​c⊤x  s.t.Gx−h∈Kd​,l≤x1​≤u,x2​∈Kp​,

where KG\mathcal{K}_GKG​ is a closed convex cone, and Kx\mathcal{K}_xKx​ is a closed convex cone.

Features

  • Multiple Cone Types Support:

    • Second-order cone (recommended but not the rsoc)

    • Rotated second-order cone

    • Exponential cone and dual exponential cone

Installation

git clone https://github.com/ZikaiXiong/PDCS.git
cd PDCS
cd src/pdcs_gpu/cuda
make # compile the cuda code, default ARCH = sm_90
cd ../../..

Usage

julia ./test/env.jl # install the dependencies
julia ./test/test_exp.jl # test the exponential cone cpu
julia ./test/test_soc.jl # test the second-order cone cpu
julia ./test/test_rsoc.jl # test the rotated second-order cone cpu

julia ./test/test_exp_gpu.jl # test the exponential cone gpu
julia ./test/test_soc_gpu.jl # test the second-order cone gpu
julia ./test/test_rsoc_gpu.jl # test the rotated second-order cone gpu

Convergence Criteria

Three criteria are considered for the performance:

  1. Primal infeasibility:

∥(Gx−h)−projKG(Gx−h)∥∞1+max⁡(∥h∥∞,∥Gx∥∞,∥projKG(Gx−h)∥∞)\frac{\|(Gx - h) - \text{proj}_{\mathcal{K}_G}(Gx - h)\|_{\infty}}{1+\max(\|h\|_{\infty}, \|Gx\|_{\infty}, \|\text{proj}_{\mathcal{K}_G}(Gx - h)\|_{\infty})}1+max(∥h∥∞​,∥Gx∥∞​,∥projKG​​(Gx−h)∥∞​)∥(Gx−h)−projKG​​(Gx−h)∥∞​​

  1. Dual infeasibility:

max⁡{∥λ1−projΛ1(λ1)∥∞,∥λ2−projKx∗(λ2)∥∞}1+max⁡{∥c∥∞,∥G⊤y∥∞}\frac{\max\{\|\lambda_1-\text{proj}_{\Lambda_1}(\lambda_1)\|_{\infty},\|\lambda_2-\text{proj}_{\mathcal{K}_x^*}(\lambda_2)\|_{\infty}\}}{1+\max\{\|c\|_{\infty},\|G^\top y\|_{\infty}\}}1+max{∥c∥∞​,∥G⊤y∥∞​}max{∥λ1​−projΛ1​​(λ1​)∥∞​,∥λ2​−projKx∗​​(λ2​)∥∞​}​

  1. Objective value accuracy:

∣c⊤x−(y⊤h+l⊤λ1++u⊤λ1−)∣1+max⁡{∣c⊤x∣,∣y⊤h+l⊤λ1++u⊤λ1−∣}\frac{|c^{\top}x-(y^{\top}h+l^{\top}\lambda_{1}^{+}+u^{\top}\lambda_{1}^{-})|}{1+\max\{|c^{\top}x|, |y^{\top}h+l^{\top}\lambda_{1}^{+}+u^{\top}\lambda_{1}^{-}|\}}1+max{∣c⊤x∣,∣y⊤h+l⊤λ1+​+u⊤λ1−​∣}∣c⊤x−(y⊤h+l⊤λ1+​+u⊤λ1−​)∣​

where λ=c−G⊤y=[λ1⊤,λ2⊤]⊤,λ1∈Λ1⊆Rn1,λ2∈Rn2\lambda=c-G^{\top}y=[\lambda_{1}^{\top},\lambda_{2}^{\top}]^{\top},\lambda_1\in \Lambda_1 \subseteq \mathbb{R}^{n_1}, \lambda_2\in \mathbb{R}^{n_2}λ=c−G⊤y=[λ1⊤​,λ2⊤​]⊤,λ1​∈Λ1​⊆Rn1​,λ2​∈Rn2​.
